highlight  

Public Instance methods

highlight(language, code = nil) {|| ...}

Highlights the passed code with the appropriate rules according to the specified language. The code can be specified either as a string or as result f a block.

Examples: 

 highlight(:ruby, 'class Test; end')

 highlight(:ruby) do
    <<-EOF
            class Test
                 end
    EOF
 end
